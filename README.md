## 1. Requirement
### 1) node 설치
- [설치 링크](https://nodejs.org/ko/download/)
- 버전: LTS(16.15.1, npm 포함)

### 2) npx 설치
```
npm install npx -g
```

### 3) 서버 실행방법
- 터미널에서 프로젝트 폴더에 접근한 후 ```npm start``` 입력을  통해 서버 실행
- 브라우저에서 3000번 포트에 localhost 열리면 서버 실행 성공
